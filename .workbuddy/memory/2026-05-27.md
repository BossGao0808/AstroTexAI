## 深度图获取来源（Komorebi 项目可用）

1. **程序合成**（最强约束）—— PIL/numpy 直接画几何体（六棱柱/方块）。1024² 灰度图，顶亮侧暗。脚本范本 `scripts/make_hex_prism_depth.py`
2. **Godot 截图 + 估计**（最适合迭代占位图）—— 用 `comfyui_controlnet_aux` 的 MiDaS / Zoe / Depth-Anything-V2 节点做单目深度估计
3. **Blender 渲染**（最准）—— 摆 3D 几何 → Z 通道直出
4. **AI 估计任意图**—— 给一张参考图，让 Depth-Anything-V2 算 depth
